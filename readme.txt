The compressed file does not include the node modules, so in order to start the project and create the website, the following steps need to be taken:

1. In the terminal, run the following command: "npm i" to install the necessary node modules.

2. Then, to run the program, type in the terminal: "npm run dev".

3. Follow the provided link, or choose preferred Browser and paste "http://localhost:5173/" into the search bar.

3. Finally to stop the programm: Press Control C and then press y.
